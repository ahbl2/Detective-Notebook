Digital Detective Guide - User Guide

Introduction

Welcome to The Digital Detective Guide! This application is designed to help detectives store, organize, and share investigative knowledge and resources.

Getting Started

First Launch

When you first launch the application:

  1. The app will create necessary directories and files
  2. A unique device ID will be generated
  3. Default categories will be created

Basic Navigation

  • Sidebar: Contains categories and main actions
  • Main Content: Displays entries for the selected category
  • Search Bar: Search across all entries
  • Add Entry Button: Create new entries

Working with Entries

Creating Entries

  1. Click the "Add Entry" button
  2. Fill in the required fields (title and category)
  3. Add optional information (description, wisdom, tags)
  4. Attach files if needed
  5. Click "Save"

Editing Entries

To edit an entry:

  1. Click the "Edit" button on the entry card
  2. Make your changes
  3. Click "Save"

Deleting Entries

To delete an entry:

  1. Click the "Delete" button on the entry card
  2. Confirm the deletion
Warning: Deleted entries cannot be recovered.

File Management

Attaching Files

You can attach files to entries:

  1. When creating or editing an entry, click "Choose File"
  2. Select the file you want to attach
  3. The file will be copied to the application's files directory

Opening Attached Files

To open an attached file:

  1. Click on the file attachment in the entry card
  2. The file will open in your system's default application

Sharing and Collaboration

Exporting Changes

To share your changes with other detectives:

  1. Click the "Export" button in the sidebar
  2. Choose a location to save the export file
  3. Share the file with other detectives

Importing Changes

To import changes from other detectives:

  1. Click the "Import" button in the sidebar
  2. Select the import file
  3. Review and confirm the changes

Rating System

The application includes a rating system to highlight valuable content:

  • Click on the stars to rate an entry (1-5 stars)
  • Each device can rate an entry once
  • Ratings are averaged across all devices

Troubleshooting

Common Issues

  • File not opening: Ensure you have the appropriate application installed
  • Import failed: Check if the import file is valid and not corrupted
  • Slow performance: The application may slow down with many entries. Consider archiving old entries.

Best Practices

  • Regularly export your changes
  • Use descriptive titles and tags
  • Keep file attachments relevant and necessary
  • Rate valuable entries to help others
  • Back up your data regularly
Note: This application is designed to work offline and from a USB drive. Always keep your data backed up and synchronized with your team.
